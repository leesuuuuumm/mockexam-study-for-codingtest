</br>

## ✔ 참여 방식
- Discode로 만나 3시간동안 2문제 solving 하는 방식으로 진행한다.
- 3시간 안에 해결하지 못한 문제는 스터디 이후에 해결하도록 한다.

- 화요일에는 그동안 못 푼 문제, 답보고 푼 문제, 피드백 받았던 문제를 각자 풀어보도록 한다.
